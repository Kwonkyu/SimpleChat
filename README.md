# SimpleChat
스프링 프레임워크 기반 채팅 애플리케이션입니다. 지난 'Need !t' 프로젝트에서 웹소켓을 처음 적용해보고 채팅 서비스를 직접 구현해보고 싶은 호기심에 진행하고 있습니다.

## 구현 기능
- [X] 주요 채팅 비즈니스 로직 구현
- [ ] HTTP 기반 채팅 서비스 구현
- [ ] 웹소켓 기반 채팅 서비스 구현

## 최종 목표
- 1대1 채팅
- 그룹 채팅
- 사진, 동영상 등 미디어 파일 첨부

## 사용 기술 스택
- Spring Boot 2.x
- Spring Data JPA
- Spring Security
- Spring Websocket
- MySQL 8.0
- H2 1.4.200
